## 常用算法的C++/Python实现
**开始时间：**
`2019/4/30`

**搭建环境：**
`Ubuntu 18.04.1  gcc 7.3.0  python 3.6`

**目录如下：**
- Sort 排序算法
	- BubbleSort 冒泡排序
	- InsertSort 插入排序
	- SelectSort 选择排序
	- MergerSort 归并排序
	- QuickSort  快速排序

- Backtrack 回溯法
	- 8Queens 8皇后问题(n皇后问题)
	- 01Bag 01背包问题
	- TSP 旅行商问题

- Graph 图算法
	- graph_matrix 图的邻接矩阵表示
	- graph_link   图的邻接链表表示
	- BFS 广度优先搜索
	- DFS 深度优先搜索
